# 🌟 Análisis Filosófico con IA

## 📌 Instrucciones

1. Abre el archivo `index.html` en tu navegador.
2. Reemplaza `"tu-api-key-aquí"` con tu clave de OpenAI.
3. Sube un archivo `.txt`, `.pdf` o `.docx`.
4. Haz clic en "Analizar con IA".
5. ¡Compara tus respuestas con el análisis de AletheIA!

## 🔐 Clave OpenAI

Ve a: https://platform.openai.com/account/api-keys
Copia tu clave y pégala en el código.
